Some Issue is appearing in the Web to resolve these issues
Right Click the Chrome Icon in Desktop
Add chrome.exe --disable-web-security --user-data-dir=c:\my-chrome-data\data to the end of target
Example: "C:\Program Files\Google\Chrome\Application\chrome.exe" --disable-web-security --user-data-dir=c:\my-chrome-data\data
Click Apply and OK
